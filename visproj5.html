Moonrise Over Mountains - Animation

This project is an intricate animation of a moonrise over mountains, capturing the subtle beauty and complexity of a nighttime landscape. It features a gradually rising moon with realistic shading and crater textures, achieved through layered gradients and shading effects that give it a 3D appearance. The stars are programmed to twinkle at randomized intervals, with subtle movements that emulate a natural night sky, while occasional shooting stars streak across, adding a dynamic touch to the otherwise tranquil scene.

A unique element of this project is the layered cloud system, where each cloud drifts independently across the sky, with transparency variations to simulate depth. The mountains are drawn with shading that softly reflects the moonlight, darkening progressively as the moon rises higher in the sky. The sky itself transitions smoothly from a dusky purple near the horizon to a deep, starlit blue, achieved through gradient blending based on the moonâ€™s position.

Through this animation, I implemented advanced procedural animation techniques, multi-layered gradient transitions, and complex object interactions that mirror real-world lighting and movement. The code is optimized to run smoothly, balancing aesthetic quality and performance for a polished final effect. View the full code and additional project details on my GitHub Repository.
