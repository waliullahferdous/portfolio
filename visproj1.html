Star War - Game

Space War Game is an exciting project that brings the classic space-shooter genre to life through an arcade-style game built in Processing. Players control a spaceship navigating through a dynamic environment, tasked with shooting moving moons while avoiding collisions. Developed as part of a structured academic course, this project was designed to reinforce key programming concepts including collision detection, real-time user input, and modular code design. The game challenges players with progressively increasing difficulty levels, enhancing both the gameplay experience and showcasing advanced coding techniques.

The game features several key components designed to enhance gameplay:

  • Spaceship Movement: Players control a spaceship that can move up and down, with visual changes indicating movement direction. This mechanic adds a layer of realism by making the spaceship appear to tilt as it moves.
  • Shooting and Collisions: The player can shoot a missile to destroy the moons. If a missile hits a moon, the player wins the round, whereas a collision with a moon or letting it reach the left side of the screen results in a loss.
  • Star-field Background: To create an immersive experience, the game includes a star-field background where stars move at different speeds to simulate depth through a parallax effect, giving the impression of zooming through space.
  • Increasing Levels of Difficulty: The game starts with three moons and progressively adds more, up to six moons, providing an escalating challenge. If the player successfully defeats all moons in the highest level, they win the game.
  • Code Structure: The code for the game is organized into well-defined functions for drawing elements like the spaceship and moons, promoting modularity and maintainability.

The Space War Game project is a demonstration of skills in interactive graphics, real-time user interaction, and game logic implementation, developed as part of an academic assignment. Due to academic integrity policies of Macquarie University, the repository containing the code and supporting documents is private.
