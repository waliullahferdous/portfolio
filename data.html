DATA & BUSINESS ANALYSIS

A collection of data-driven projects demonstrating analytical expertise and business insights using Python, SQL, R, Tableau, Power BI, and Excel.

Overview

The Data & Business Analysis section showcases a range of data-driven projects utilizing tools such as Python, SQL, R, Tableau, Power BI, and Excel. These projects highlight my expertise in data cleaning, exploratory analysis, feature selection, and predictive modeling, along with business intelligence tasks such as dashboard creation, reporting, and the use of appropriate graphical visualizations. Each project demonstrates my ability to uncover insights, analyze trends, and provide actionable recommendations, illustrating a strong command of data analysis techniques essential for solving real-world business problems.

Data Science Portfolio Project at Macquarie University

This project includes python notebook files for the COMP2200/6200 Portfolio task (S1 2022) at Macquarie University. It analyzes "MovieLens" data from GroupLens Research and "StudentsPerformance" data from Kaggle, testing fundamental data science skills in data visualization, predictive modeling, evaluation, regression, classification, clustering, and other machine learning techniques.

2023 Sales Performance Analysis and Strategic Insights for MTC

This project is a business report analyzing the 2023 sales performance of Mega Transportation Company (MTC), focusing on strategic insights for market expansion.
