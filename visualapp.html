VISUAL APPLICATIONS & GAMES

Full-stack web applications built with the MERN stack, designed for responsive, user-friendly, and scalable solutions.

Overview

The Web Development section highlights my experience building dynamic, full-stack applications using the MERN stack (MongoDB, Express, React, and Node.js). These projects showcase my skills in creating responsive, user-friendly websites that integrate robust back-end functionality with engaging front-end design. Each project reflects my commitment to delivering seamless web solutions that are efficient, scalable, and tailored to user needs.

Star War - Game

An interactive game developed using Processing, where players control a spaceship navigating through space to shoot moving moons while avoiding collisions. The game incorporates multiple levels, increasing difficulty with additional moons, and features a star-field background for an immersive experience.

Pathfinder - A-Star Algorithm

This project focused on developing a pathfinding algorithm using the A* search method to control an unmanned vehicle navigating through obstacles. Implemented in Java Processing, the project demonstrates real-time pathfinding with graphical feedback, allowing the vehicle to find optimal routes to various targets on a grid.

  • Learn more

Mario - Game

This project is a Mario-style game developed in Processing, featuring classic platforming elements such as collecting coins, avoiding obstacles, and interacting with enemies. The game also includes dynamic environmental effects like day-night cycles, rain, and moving background elements to create an engaging player experience.

  • Learn more

Sunset Over Ocean - Animation

This project is an animation of a sunset over an ocean, created using Processing. It features dynamic elements such as a descending sun, rolling waves with foam, a gradient sky transitioning from day to sunset, and moving clouds that change color as the sun sets.

  • Learn more

Moonrise Over Mountains - Animation

This is an animated moonrise over mountains with realistic shading, twinkling stars, and occasional shooting stars across a gradient night sky. Floating clouds and subtly lit mountain peaks add depth to the scene, enhancing the serene nighttime atmosphere. The moon, stars, and clouds all move dynamically to create a lifelike effect.

  • Learn more

Hypnotic Flower Bloom - Animation

This project creates a mesmerizing animation of a blooming flower with hundreds of petals that dynamically rotate, grow, and change color in a hypnotic pattern. Each petal's shape and movement are generated using polar coordinates and sinusoidal curves, producing a flowing, organic effect.
