Fractal Galaxy Spiral - Animation

The Fractal Galaxy Spiral project is a mesmerizing animation of a rotating galaxy, built with 500 particles arranged in a spiral pattern inspired by fractal geometry. Using polar coordinates and fractal scaling, each particle moves dynamically to create the appearance of a swirling galaxy, with particles radiating out from the center in a graceful, rotating motion. The spiral structure is carefully controlled by parameters that adjust the tightness of the rotation, creating a cosmic effect that mimics the structure of real galaxies.

A major feature of this project is the color gradient applied to each particle, ranging from deep purple to vibrant blue and finally to white. This color transition gives the animation a sense of depth and movement, as particles appear to shift and glow as they move along the spiral path. A subtle oscillating wave effect further enhances the particles' motion, giving them a slight "wobble" that makes the galaxy appear alive and ever-evolving.

The animation combines principles of fractal geometry and smooth, continuous rotation to create a visually captivating galaxy that feels both intricate and otherworldly. Explore the code and learn more on my GitHub Repository.
